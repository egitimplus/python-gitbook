# Methods

```python
my_list = ['orange', 'banana', 'apple', 'cherry']

# count
print(my_list.count('orange'))

# index
print(my_list.index('orange'))

# reverse

print(list(reversed(my_list)))

print(my_list[::-1])

my_list.reverse()
print(my_list)
```
