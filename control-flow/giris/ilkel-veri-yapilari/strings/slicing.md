# Slicing

```python
text = "Hello World Again"

print(text[0:5])
print(text[:5])

print(text[6:11])
print(text[-11:-6])
print(text[6:-6])

print(text[6:])

print(text[1:-1:2])
print(text[::2])

print(text[::-1])
```
