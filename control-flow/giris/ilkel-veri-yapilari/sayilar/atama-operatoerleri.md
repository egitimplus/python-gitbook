# Atama Operatörleri

|           |       |
| --------- | ----- |
| Ekleme    | +=    |
| Çıkarma   | -=    |
| Çarpma    | \*=   |
| Kuvvet    | \*\*= |
| Bölme     | /=    |
| Tam Bölme | //=   |
| Kalan     | %=    |

```python
x = 9

x += 3 # Addition
print(x)

x -= 5 # Substraction
print(x)

x *= 2 # Multiplication
print(x)

x **= 2 # Exponentiation
print(x)

x /= 3 # True Division
print(x)

x //= 10 # Floor Division
print(x)

x %= 4  # Modulus
print(x)
```

```

----------------------------------------
12
7
14
196
65.33333333333333
6.0
2.0
```
