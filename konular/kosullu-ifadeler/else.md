# Else

{% hint style="info" %}
```
if <condition>:
    ...
    ...
else:
    ...
    ...
```
{% endhint %}

```python
x = 3
if x > 5:
    print('x is greater than 5')
else:
    print('x is less than 5')

# Output: x is less than 5
```
