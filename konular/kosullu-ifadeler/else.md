# Else

{% hint style="success" %}
```
if <condition>:
    ...
    ...
else:
    ...
    ...
```
{% endhint %}

```python
x = 3
if x > 5:
    print('x is greater than 5')
else:
    print('x is less than 5')
```

<pre><code><strong>Output:
</strong>----------------------------------------
x is less than 5</code></pre>
