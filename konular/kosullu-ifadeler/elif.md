# Elif

{% hint style="info" %}
```
if <condition>:
    ...
    ...
elif <condition>:
    ...
    ...
else:
    ...
    ...
```
{% endhint %}

```python
x = 4
if x > 5:
    print('x is greater than 5')
elif x > 3:
    print('x is greater than 3')
else:
    print('x is less than 5')
    
# Output: x is greater than 3
```
