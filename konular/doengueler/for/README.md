# For

{% hint style="success" %}
```
for variable in iterable:
    ...
    ...
    ...
```
{% endhint %}

{% hint style="info" %}
python kelimesinin harflerini yazdırınız
{% endhint %}

```python
for letter in 'Python':
    print(letter)
    
'''

y
t
h
o
n
'''
```

{% hint style="info" %}
