# tip değiştirme

{% tabs %}
{% tab title="int()" %}
{% hint style="info" %}
int(): tam sayıya çevirmek için kullanılır.
{% endhint %}

```python
x = int('123123123')
print(type(x)) # <class 'int'>
```
{% endtab %}

{% tab title="float()" %}
{% hint style="info" %}
float(): ondalıklı sayıya çevirmek için kullanılır.
{% endhint %}

```python
x = float(5)
print(type(x)) # <class 'float'>
```
{% endtab %}

{% tab title="str()" %}
{% hint style="info" %}
str() : metine çevirmek için kullanılır.
{% endhint %}

```python
x = str(23.25)
print(type(x)) # <class 'str'>
```
{% endtab %}

{% tab title="complex()" %}
{% hint style="info" %}
complex(): karmaşık sayıya çevirmek için kullanılır.
{% endhint %}

```python
x = complex(5)
print(type(x)) # <class 'complex'>
```
{% endtab %}
{% endtabs %}
