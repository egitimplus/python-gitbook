# Yorumlar

{% tabs %}
{% tab title="single line" %}
{% hint style="info" %}
tek satırlık yorumlar
{% endhint %}

```python
# This is a comment
print("Hello, World!")

# Output: Hello, World!
```
{% endtab %}

{% tab title="inline" %}

```python

```
{% endtab %}

{% tab title="multiline" %}

