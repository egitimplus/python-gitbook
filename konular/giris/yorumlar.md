# Yorumlar

## tek satırlık yorumlar

```python
# This is a comment
print("Hello, World!")

# Output: Hello, World!
```

## satır içi yorumlar

```python
print("Hello, World!") #This is a inline comment
```

## çok satırlık yorumla

```python
"""
This is a
multiline
comment
"""
print("Hello, World!")

# Output: Hello, World!

```
