# Yorumlar

## Tek satırlık yorumlar

```python
# This is a comment
print("Hello, World!")

# Output: Hello, World!
```

## Satır içi yorumlar

```python
print("Hello, World!") #This is a inline comment
```

## Çok satırlık yorumla

```python
"""
This is a
multiline
comment
"""
print("Hello, World!")

# Output: Hello, World!

```
