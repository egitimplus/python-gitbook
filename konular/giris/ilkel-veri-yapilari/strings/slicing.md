# Slicing

```python
text = "Hello World Again"

print(text[0:5])
# Output : Hello

print(text[:5])
# Output : Hello

print(text[6:11])
# Output : World

print(text[-11:-6])
# Output : World

print(text[6:-6])
# Output : World

print(text[6:])
# Output : World Again

print(text[1:-1:2])
# Output : el ol gi

print(text[::2])
# Output : HloWrdAan

print(text[::-1])
# Output : niagA dlroW olleH
```
