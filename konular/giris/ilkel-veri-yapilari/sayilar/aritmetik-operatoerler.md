# Aritmetik Operatörler

| Operatör | Açıklama    | Örnek    | Sonuç |
| -------- | ----------- | -------- | ----- |
| +        | Toplama     | 2 + 2    | 4     |
| -        | Çıkarma     | 5 - 2    | 3     |
| \*       | Çarpma      | 3 \* 5   | 15    |
| /        | Bölme       | 22 // 8  | 2.75  |
| \*\*     | Üs alma     | 2 \*\* 3 | 8     |
| %        | Mod / Kalan | 22 % 8   | 6     |
| //       | Tam Bölen   | 22 // 3  | 2     |

```python
# Addition
print(5 + 3)      # 8 

# Substraction
print(6 - 1)      # 5 

# Multiplication 
print(4 * 4)      # 16

# Exponentiation 
print(2 ** 4)     # 16 

# Floor Division
print(7 / 2)      # 3.5

# True Division
print(15 // 2)    # 7

# Modulus 
print(22 % 4)     # 2
```

```python
# İki metini toplarken dikkat etmek lazım!
print('5' + '3') # 53

# Bir metin ve tam sayıyı çarparken dikkat etmeliyiz.
print('4' * 4)    # 4444
```
