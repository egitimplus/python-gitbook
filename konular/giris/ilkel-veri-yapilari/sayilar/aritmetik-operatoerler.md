# Aritmetik Operatörler

| İşlem                      | Operatör |
| -------------------------- | -------- |
| Ekleme (Addition)          | +        |
| Çıkarma (Substraction)     | -        |
| Çarpma (Multiplication)    | \*       |
| Kuvvet (Exponentiation)    | \*\*     |
| Bölme (True Division)      | /        |
| Tam Bölme (Floor Division) | //       |
| Kalan (Modulus)            | %        |

```python
print(5 + 3)     # 8 
print(6 - 1)     # 5 
print(4 * 4)     # 16 
print(2 ** 4) # 16 
print(6 / 2) # 3.0
print(15 // 2) # 7
print(22 % 4) # 2
```
