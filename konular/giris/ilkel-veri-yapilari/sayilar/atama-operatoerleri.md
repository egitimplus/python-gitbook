# Atama Operatörleri

|                            |       |
| -------------------------- | ----- |
| Ekleme (Addition)          | +=    |
| Çıkarma (Substraction)     | -=    |
| Çarpma (Multiplication)    | \*=   |
| Kuvvet (Exponentiation)    | \*\*= |
| Bölme (True Division)      | /=    |
| Tam Bölme (Floor Division) | //=   |
| Kalan (Modulus)            | %=    |

```python
x = 9

x += 3  
print(x) # 12

x -= 5  
print(x) # 7

x *= 2 
print(x) #  14

x **= 2  
print(x) # 196
 
x /= 3 
print(x) # 65.33333333333333

x //= 10 
print(x) # 6.0

x %= 4 
print(x) # 2.0
```
