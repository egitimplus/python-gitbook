# Atama Operatörleri

| +=    | Toplama     |
| ----- | ----------- |
| -=    | Çıkarma     |
| \*=   | Çarpma      |
| /=    | Bölme       |
| \*\*= | Üs alma     |
| %=    | Mod / Kalan |

```python
# Addition

x += 3  
print(x)     # 12

# Substraction
x -= 5  
print(x)     # 7

# Multiplication 
x *= 2 
print(x)     #  14

# Exponentiation 
x **= 2 
print(x)     #  196

# Floor Division
x /= 4
print(x)     # 49.0

# True Division
x //= 3     # 16

# Modulus 
x %= 5     # 1
```
