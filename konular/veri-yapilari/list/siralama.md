# Sıralama

```python
# sort() metodu ile listeleri sıralayabiliriz.

my_list = ['orange', 'banana', 'apple', 'cherry']
my_list.sort()
print(my_list)

my_list.sort(reverse=True)
print(my_list)
```

```
['apple', 'banana', 'cherry', 'orange']
['orange', 'cherry', 'banana', 'apple']
```
