# Atama Operatörleri

| Operatör | Açıklama    |
| -------- | ----------- |
| +=       | Toplama     |
| -=       | Çıkarma     |
| \*=      | Çarpma      |
| /=       | Bölme       |
| \*\*=    | Üs alma     |
| %=       | Mod / Kalan |

```python
x += 3      # Addition
print(x)    # 12

x -= 5      # Substraction
print(x)    # 7

x *= 2      # Multiplication 
print(x)    # 14

x **= 2     # Exponentiation 
print(x)    # 196

x /= 4      # Floor Division
print(x)    # 49.0

x //= 3     # True Division
print(x)    # 16

x %= 5      # Modulus 
print(x)    # 1
```
